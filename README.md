# dms
毕业设计-基于VaadinFlow+SpringBoot的宿舍管理系统(DormitoryManagementSystem)
功能：
1、宿舍空间管理
2、房客管理
3、收费记录管理

#环境要求
Java8+ + maven3.6

#技术栈
Spring Boot 2.x，Spring Data Jpa，Vaadin Flow
